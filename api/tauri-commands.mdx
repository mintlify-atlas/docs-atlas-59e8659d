## Search & Browse

### search

Search screenshots by OCR text using the daemon's Search D-Bus method.

<ParamField path="query" type="string" required>
  Search query (supports FTS5 syntax: AND, OR, NOT, phrases)
</ParamField>

<ParamField path="filters" type="UiSearchFilters" required>
  Search filters object
  <Expandable title="UiSearchFilters">
    <ParamField path="start_time" type="i64 | null">
      Filter after timestamp
    </ParamField>
    <ParamField path="end_time" type="i64 | null">
      Filter before timestamp
    </ParamField>
    <ParamField path="app_name" type="string | null">
      Filter by app name
    </ParamField>
    <ParamField path="limit" type="i64" default="50">
      Max results
    </ParamField>
    <ParamField path="offset" type="i64" default="0">
      Pagination offset
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="response" type="SearchResponse">
  Search results with deduplication metadata
</ResponseField>

**Source:** `lib.rs:70-99`

<CodeGroup>
```typescript Frontend Usage
import { invoke } from '@tauri-apps/api/core';

const results = await invoke('search', {
  query: 'PostgreSQL connection pool',
  filters: {
    start_time: null,
    end_time: null,
    app_name: null,
    limit: 50,
    offset: 0,
  },
});
```
</CodeGroup>

### ask

Send a message to the AI chat assistant. Streams response via `ask-token`, `ask-done`, and `ask-error` events.

<ParamField path="session_id" type="string" required>
  Session ID from ask_new_session
</ParamField>

<ParamField path="message" type="string" required>
  User message
</ParamField>

<ResponseField name="response" type="AskResponse">
  Immediate response with session ID and screenshot references
  <Expandable title="AskResponse Fields">
    <ResponseField name="session_id" type="string" required />
    <ResponseField name="references" type="ScreenshotReference[]" required>
      Screenshots used as context
    </ResponseField>
  </Expandable>
</ResponseField>

**Events emitted:**
- `ask-token` — Streaming response tokens (`AskTokenPayload`)
- `ask-done` — Final response (`AskDonePayload`)
- `ask-error` — Error during generation (`AskErrorPayload`)

**Source:** `lib.rs:685-1008`

<CodeGroup>
```typescript Frontend Usage
import { invoke, listen } from '@tauri-apps/api/core';

const sessionId = await invoke('ask_new_session');

const unlisten = await listen('ask-token', (event) => {
  console.log('Token:', event.payload.token);
});

const { references } = await invoke('ask', {
  sessionId,
  message: 'What did I work on this morning?',
});
```
</CodeGroup>

## Type Definitions

All request and response types are defined in `rewindos-core/src/schema.rs`. Key types:

- `SearchResponse` — Search results with deduplication metadata
- `SearchFilters` — Search query filters
- `Screenshot` — Screenshot metadata
- `ActivityResponse` — Hourly/daily activity stats
- `Bookmark` — Bookmark metadata
- `Collection` — Screenshot collection
- `JournalEntry` — Journal entry
- `JournalTag` — Tag metadata
- `JournalTemplate` — Journal template
- `DaemonStatus` — Daemon status

Refer to the source code for detailed type definitions.
